# CDN

使用 Github，jsDelivr，搭建一个轻量靠谱的 CDN。

## 原理

Github 做 CDN 存储，jsDelivr 做 CDN 服务器。

## 效果

jsDelivr 节点遍布全球，有中国节点。

## 统计

jsDelivr 提供了一个可以看 CDN 文件和使用流量的地址：<https://www.jsdelivr.com/package/gh/likebeta/cdn> 。
